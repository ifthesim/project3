User Story
1. The user will be able to sign up.
2. The user will be able to sign in and create new pin.
3. The user will be able to sign in and post image with description.
4. The users will be able to vote on certain pins that they like.
5. The user will be able to update or delete their pin by signing in.
6. The user will be able to share the app on Facebook and twitter.
